# NVDA for SiteGist

`NVDA 2020.2` screen reader for Windows. 

## Download

1. Goto [this page](./files/nvda.zip) and click `download` (nvda.zip, 76 MB)
   or just
   ```
   git clone https://github.com/ruslan-rv-ua/nvda4sitegist
   ```
   
1. Unzip `nvda.zip`


## Quckstart

To run NVDA just start `nvda.exe`.
